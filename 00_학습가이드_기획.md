# CMOS_SS26 학습 가이드 제작 기획

이 문서는 `Lecture/`, `Exercise/`, `Assignment/` 자료를 기준으로 만든 국문 학습 노트의 설계도이다. 행정 안내, 교수진 소개, 제출 방식처럼 시험 이해에 직접 필요하지 않은 내용은 제외하고, CMOS 소자, 전력, 신뢰성, 표준셀, SPICE 실습, RTL-to-GDS 물리 설계를 큰 흐름으로 묶었다. Assignment 가이드는 제출 절차보다 문제의 의도, 보고서 결과의 타당성, 이론적 해석에 집중한다.

## 목표

이 과목의 핵심 목표는 CMOS 칩 설계를 "소자 물리 - 회로 동작 - 표준셀 - 신뢰성 - 실습 측정"의 한 흐름으로 이해하는 것이다.

강의에서 반복되는 중심 질문은 다음과 같다.

- MOSFET은 어떻게 켜지고, 전류는 무엇에 의해 결정되는가?
- FinFET과 nanosheet는 planar MOSFET의 어떤 한계를 해결하려고 등장했는가?
- 전압, 온도, 문턱전압, 이동도, 부하 커패시턴스는 성능과 전력에 어떻게 연결되는가?
- aging은 어떤 물리 현상으로 생기며, 회로 지연과 오류로 어떻게 이어지는가?
- 표준셀은 왜 디지털 칩의 기본 단위이며, delay/power library는 어떻게 만들어지는가?
- aging을 견디기 위해 넣는 guardband를 recovery, annealing, aging-aware synthesis로 어떻게 줄일 수 있는가?
- SPICE/HSPICE 실습에서 이 이론들을 어떤 측정값으로 확인하는가?
- RTL/gate-level netlist는 placement, routing, metal stack을 거쳐 어떻게 layout이 되는가?
- I/O cell, setup/hold timing, transistor-level layout은 완성된 chip에서 어떻게 연결되는가?

## 작성 원칙

- 수식은 외우는 대상이 아니라 "어떤 변수가 어느 방향으로 작용하는지"를 먼저 설명한다.
- 설명은 쉬운 난이도로 쓰되, 부정확한 비유는 쓰지 않는다.
- 그래프는 축, 증가/감소 방향, 물리적 이유, 회로적 결과 순서로 해석한다.
- Exercise는 도구 사용법 자체보다 "어떤 이론을 어떤 측정값으로 확인하는가"에 초점을 둔다.
- 시험 대비를 위해 각 장 끝에 핵심 체크리스트를 둔다.

## 전체 학습 순서

1. 01에서 기술 scaling 한계와 MOSFET 구조 진화를 먼저 이해한다.
2. 02에서 MOSFET 전류, 전력, 온도 효과를 수식으로 연결한다.
3. 03에서 신뢰성 저하가 왜 생기고, 왜 시간이 지나며 회로가 느려지는지 이해한다.
4. 04에서 HSPICE netlist와 `.measure` 명령으로 이론을 측정값으로 바꾸는 법을 익힌다.
5. 05에서 표준셀이 디지털 칩 설계의 기본 단위가 되는 이유를 정리한다.
6. 06에서 aging을 표준셀/경로/칩 설계 수준에서 보정하는 guardband 개념을 학습한다.
7. 07에서 recovery/annealing과 aging-aware synthesis로 guardband 비용을 줄이는 방법을 학습한다.
8. 08에서 aging이 $V_{th}$뿐 아니라 mobility, subthreshold slope, $g_{m}$, capacitance, power, delay까지 바꾼다는 점을 학습한다.
9. 09에서 circuit error가 항상 fault가 아니며, masking과 approximate computing으로 guardband를 quality trade-off로 바꿀 수 있음을 이해한다.
10. 10에서 AOI21과 ripple-carry adder의 구조, rise/fall 비대칭, path sensitization을 이해하고 Python으로 path/power case를 관리한다.
11. 11에서 RTL/gate-level netlist가 placement/routing을 거쳐 GDSII로 바뀌는 흐름과 I/O ring, setup/hold STA, transistor-level cell layout, DRC/LVS signoff를 함께 익힌다.
12. 12–17에서 같은 원리를 실제 Assignment 결과에 다시 적용한다. 순서는 DC 소자 특성 → aging → NAND characterization/guardband → AOI21/adder → inverter sizing → layout/extraction이다.
